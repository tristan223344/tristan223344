My opinion of the game

my opinion on the game is that it is a good game overall but,
it has some bugs and glitches. for instance sometimes when you try to load
the game gets stuck on the loading screen. although the game is in a
somewhat late-stage and the bugs and glitches are not getting any better.
On the positive side of the game, it overall is a 9/10 in my opinion due
the fact that Visual Concepts one of the creators of the game does listen to
the community and change and fix some of the glitches that make the game
near to unplayable. With 2k23 soon to be out, I would hope that these glitches are fixed.
after all the game does have a place where you can leave a comment or get help, but it
takes anywhere between 1-3 weeks to even get your message noticed.
